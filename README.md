House Price Prediction
This project implements a machine learning solution to estimate real estate prices. 
By analyzing various property features, the model provides data-driven valuations to assist in real estate decision-making.
Project Overview
The objective is to identify the key factors that drive property values and build a robust regression model.
The project covers the full workflow, from exploratory data analysis to a functional prediction interface.

File Structure
 * house_predict.csv: The dataset containing property specifications and historical pricing.
 * House_predict.ipynb: Jupyter notebook with the complete data cleaning, analysis, and model training code.
 * model_h.pkl: The trained machine learning model saved for deployment.
 * App_House_Predict.py: A Python script to run the interactive web application.
 * Requirements.txt: List of Python dependencies required to run the project.

How to Run
 * Clone the repository:
   git clone https://github.com/MohamedElsayedElbadwy/house_price_prediction.git

 * Install dependencies:
   pip install -r Requirements.txt

 * Start the app:
   streamlit run App_House_Predict.py

Key Features
 * Comprehensive analysis of structural and geographical price factors.
 * Accurate valuation using optimized regression algorithms.
 * Simple and intuitive web dashboard for real-time estimates.
